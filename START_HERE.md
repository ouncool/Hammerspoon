## ✨ 快速参考

### 全局快捷键
- `Hyper + G` - 打开浏览器
- `Hyper + T` - 打开终端
- `Hyper + F` - Finder 目录在终端打开
- `Hyper + V` - Finder 目录在 VS Code 打开
- `Hyper + R` - 进入窗口管理模式

### 窗口管理 (`Hyper + R`)
- `h/l` - 左/右半屏
- `y/u/i/o` - 四分位
- `f` - 全屏
- `c` - 关闭
- `q/Esc` - 退出
