# 快速参考 - 图片压缩功能

## 使用步骤

```
1. 复制图片到剪切板
   ↓
2. 按 Option + R 进入 Vim 模式
   ↓
3. 快速按两次 i 键 (ii)
   ↓
4. 等待提示 "✅ 图片已压缩并复制到剪切板"
   ↓
5. 粘贴压缩后的图片 (Cmd + V)
```

## Vim 模式完整快捷键

| 快捷键 | 功能 |
|--------|------|
| **ii** | **压缩剪切板图片** |
| h | 左半屏 |
| l | 右半屏 |
| j | 下半屏 |
| k | 上半屏 |
| y | 左上四分 |
| u | 左下四分 |
| i | 右上四分 |
| o | 右下四分 |
| f | 最大化窗口 |
| c | 关闭窗口 |
| tab | 显示帮助 |
| q / Esc | 退出 Vim 模式 |

## 注意事项

- 第一次按 `i` 后有 0.5 秒的等待时间来判断是否按第二个 `i`
- 如果只按一次 `i`，会执行右上四分窗口调整
- 压缩过程中会显示 "正在处理图片..." 提示
- 支持 PNG、JPEG、TIFF 等常见格式的剪切板图片
- 压缩质量设置为 75（可在 `image-compressor.lua` 中修改）
